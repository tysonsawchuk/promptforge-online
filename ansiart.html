What is ANSI/ASCII Art?

      

        ASCII art is digital artwork created using the ASCII character set—letters, symbols, and numbers—to form images, designs, and animations. ANSI art expands on this by adding color, custom palettes, and control codes, making it famous in BBS (Bulletin Board System) and early Internet culture. Modern ANSI art includes Unicode, emoji, and all creative hacks that use text as graphics.
      

      
ANSI/ASCII Art Glossary

      
    
        
  • ASCII: American Standard Code for Information Interchange; original character set for plain text.
    • 
        
  • ANSI: Extended ASCII with color/control codes (e.g., ESC[31m for red text).
    • 
        
  • Palette: Set of colors used in an ANSI image (BBS, CGA, ICE, custom, etc).
    • 
        
  • Dither: Technique to simulate shades using patterns of characters or colors.
    • 
        
  • Density: How many characters are used to render an image (higher = more detail).
    • 
        
  • Block Font: A style that uses “blocky” characters for headlines/titles.
    • 
        
  • Monospaced: Every character has the same width; essential for ANSI/ASCII art to align.
    • 
        
  • BBS Art: Art made for bulletin board systems (pre-web online communities).
    • 
        
  • Unicode Art: Uses modern Unicode symbols, emojis, or non-ASCII glyphs.
    • 
        
  • ICE Colors: An oldschool 16-color palette popular in the ANSI scene.
    • 
        
  • GIF ANSI: Animated ANSI art saved as a GIF (animated “frame-by-frame”).
    • 
        
  • Remix: Editing or restyling an existing ANSI/ASCII work in a new way.
    • 
        
  • SVG Export: Vector version of ANSI/ASCII, scales perfectly.
    • 
        
  • Scrolling Banner: Horizontal animation—classic in text terminals.
    • 
        
  • Matrix Rain: Animation effect inspired by The Matrix (falling characters).
    • 
        
  • ANSI Animation: Any moving ASCII/ANSI image (animated effects, video, GIF, etc).
    • 
      

      
History & Culture

      

        ANSI and ASCII art date back to early mainframe computers and were especially popular in 1980s/90s BBS communities. Artists (called “sceners”) would compete in “artpacks,” pushing the limits of characters and color. Many classic works are preserved at Sixteen Colors and asciiart.eu.
