FAQ

        

          
Instant clean ASCII? Image → ASCII with Fine ramp, width 140–180, line‑height 1.9, gamma 1.25, and Bayer 8×8 for crisp edges.

          
Video looks coarse? Up the width & FPS, pick “Fine” or “Braille smooth”, then lift gamma slightly.

          
FIGlet fonts? 50+ fonts. Add outline/shadow; swap the fill glyph for themed posters.

          
ANSI gradients? Toggle HTML Preview to see colors in the browser; copy ANSI for terminals.

          
Exports? Save PNG/TXT per tool or batch via Export. PNG font size controls crispness.
